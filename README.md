# 🚀 Java Security & Hello Controller Project

Это учебный проект на языке **Java**, демонстрирующий навыки работы с объектно-ориентированным программированием (ООП), потоками данных (Stream API) и системой контроля версий Git.

## 🛠 Технологии в проекте
* **Язык:** Java 11+
* **Инструменты:** GitHub Web Interface, Git
* **Ключевые концепции:**
    * Инкапсуляция и статические вложенные классы.
    * Обработка коллекций через `java.util.stream`.
    * Работа с датой и временем (`java.time`).
    * Имитация бизнес-логики контроллера (Controller Pattern).

---

## 📂 Структура кода
Проект содержит основной класс `HelloController`, который включает в себя:
* **User Database:** Имитация хранилища пользователей через `ArrayList`.
* **Security Checks:** Валидация прав доступа и фильтрация активных пользователей.
* **Logging System:** Внутренний сервис для отслеживания системных событий.

---

## 🚀 Как запустить проект
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com
   ```
2. Скомпилируйте файл:
   ```bash
   javac HelloController.java
   ```
3. Запустите программу:
   ```bash
   java HelloController
   ```

## 📈 История разработки (Commits)
Проект создавался поэтапно для демонстрации навыков работы с Git:
1. **Initial Commit:** Настройка структуры проекта.
2. **Add Logic:** Реализация базового функционала приветствия.
3. **Refactoring:** Оптимизация кода с использованием Stream API и добавление логгера.
